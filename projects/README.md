# Projects
I've always enjoyed building things.

Sometimes it's software, sometimes it's hardware, and occasionally it's something that probably didn't need to exist but seemed like a fun idea at the time. Whether it's a custom PC, a web application, an Arduino project, or something running in my homelab, I enjoy the process of taking an idea and turning it into something real.

Most of these projects start out as a way to learn something new or solve a problem I've run into myself. Along the way I usually discover a few mistakes, a few better ways of doing things, and occasionally something worth writing about.

This section is a collection of those projects. Some are finished, some are still evolving, and a few may never quite be complete. Regardless, they all taught me something and hopefully you'll find something useful here too.
